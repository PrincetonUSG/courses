# precourser

Apologies for lack of real documentation, please reach out to mshu@alumni.princeton.edu for help.

## setup

- install node, npm
- in the root directory create a .env file that looks like
```
MONGODB_URI=<database uri>
DEV_USER=<netid>
```

## to run

### precourser server
```
node app.js
```

### react development server
```
cd client
npm start
```
